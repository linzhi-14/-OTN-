**【必     看      常      见     错    误     解    决】**

第一个错误：

安装结束后，直接点击启动即可，若您出现了报错的情况

那可能是 【Super Resolution】这个优化模组导致的

原因是可能您的电脑GPU过于落后，无法搭载该模组

若您出现了该错误，可在【模组管理】里面将该模组删掉即可，不会影响游戏多少，帧数会低一点而已！

第二个错误：

若您打开世界后发现屏幕只有一片黑色，什么都看不清楚

这说明遥远的地平线与您的电脑产生冲突

只需要将【distant horizon】禁用或删除即可！
